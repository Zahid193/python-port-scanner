# 🔍 Python Port Scanner

## 📌 Overview:

This project is a simple cybersecurity tool built using Python that scans a target system to identify open ports and their associated services.

It demonstrates fundamental concepts of:

* Network communication
* Port scanning
* Basic cybersecurity practices

---

## 🚀 Features:

* Scans common ports (FTP, SSH, HTTP, HTTPS, etc.)
* Displays open ports with service names
* Saves scan results to a file
* Measures scan time
* Handles invalid inputs

---

## 🛠️ Technologies Used:

* Python
* Socket Programming
* Basic Networking Concepts

---

## ▶️ How to Run:

```bash
python port_scanner.py
```

Then enter:

* Target domain/IP (e.g., scanme.nmap.org)

---

## 📊 Example Output:

```
[OPEN] Port 22 (SSH)
[OPEN] Port 80 (HTTP)
```
